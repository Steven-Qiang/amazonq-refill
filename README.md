# Amazon Q Refill

Amazon Q 多账号登录管理工具。

## 下载

前往 [Releases](https://github.com/Steven-Qiang/amazonq-refill/releases) 下载编译好的安装包。

## 功能

- 多账号管理, 方便续杯
- 独立浏览器会话
- 邮箱验证码自动接收

## 使用说明

1. 添加账号：填写邮箱、密码、POP3 服务器信息（端口必须是 995）
2. 选择账号后点击"开始登录"
3. 在弹出的对话框中复制账号密码
4. 在浏览器窗口完成登录
5. 点击"完成登录"保存会话

## 邮箱配置

常见邮箱 POP3 服务器（端口 995）：

- 163 邮箱: `pop.ym.163.com`
- QQ 邮箱: `pop.qq.com`
- Gmail: `pop.gmail.com`
- Outlook: `outlook.office365.com`

注意：需要在邮箱设置中开启 POP3 服务并使用授权码。
